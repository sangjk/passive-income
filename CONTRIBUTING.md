# Contributing to PassiveIncome Hub

Thank you for your interest in contributing! This project thrives because of community members like you.

---

## Ways to Contribute

### 1. Submit a Strategy Guide

Share your proven side hustle strategy with the community.

#### Guide Template

```markdown

[Brief description of the strategy and why it works]

| Detail | Info |
|--------|------|
| **Difficulty** | ⭐–⭐⭐⭐⭐ |
| **Time to First $** | [Timeframe] |
| **Earning Potential** | $[Range]/month |
| **Upfront Investment** | $[Amount] |
| **Best Platforms** | [Platforms] |

---

## Step 1: [First Step]

[Detailed instructions]

## Step 2: [Second Step]

[Detailed instructions]

## Step 3: [Third Step]

[Detailed instructions]

## Step 4: [Fourth Step]

[Detailed instructions]

## Step 5: [Fifth Step]

[Detailed instructions]

---

## 🤖 AI Augmentation

| Task | AI Tool | How to Use |
|------|---------|------------|
| [Task] | [Tool] | [Instructions] |

---

## 💰 Income Breakdown

| Level | Monthly Income | Hours/Week | Strategy |
|-------|---------------|------------|----------|
| **Beginner** | $[Range] | [Hours] | [Strategy] |
| **Intermediate** | $[Range] | [Hours] | [Strategy] |
| **Advanced** | $[Range] | [Hours] | [Strategy] |

---

## ⚠️ Common Mistakes

1. [Mistake 1]
2. [Mistake 2]
3. [Mistake 3]
4. [Mistake 4]
5. [Mistake 5]
```

#### Guide Requirements

- **Real experience** — Only submit strategies you've personally tried or thoroughly researched
- **Specific numbers** — Include actual income ranges, timeframes, and pricing
- **Actionable steps** — Every step should be something the reader can DO, not just think about
- **AI augmentation** — Include how AI tools can help with each strategy
- **Common mistakes** — Share what doesn't work, not just what does
- **Proper formatting** — Follow the template structure

### 2. Improve Existing Guides

Found outdated information or have a better approach?

1. Open an issue describing what needs updating
2. Or submit a pull request with the improvements

### 3. Share Your Success Story

Inspire others by sharing your journey.

1. Copy the story template from [success-stories.md](community/success-stories.md)
2. Fill in your experience with real numbers
3. Submit as a pull request to `community/stories/`

### 4. Add Resources

Know a great tool, template, or resource?

1. Add it to the appropriate file in `resources/`
2. Include: name, purpose, cost, and a brief description
3. Only include tools you've actually used

## Submission Process

### Step 1: Fork & Clone

```bash
git clone https://github.com/yourusername/PassiveIncome-Hub.git
cd PassiveIncome-Hub
```

### Step 2: Create a Branch

```bash
git checkout -b your-username/your-contribution
```

### Step 3: Make Your Changes

- Follow the existing file structure
- Use the templates provided
- Keep formatting consistent

### Step 4: Commit & Push

```bash
git add .
git commit -m "Add: [description of your contribution]"
git push origin your-username/your-contribution
```

### Step 5: Open a Pull Request

- Use a clear, descriptive title
- Reference any related issues
- Describe what you've added or changed

## Content Guidelines

### Do

- ✅ Share strategies you've personally validated
- ✅ Include specific, realistic numbers
- ✅ Write clear, actionable steps
- ✅ Cite sources when making claims
- ✅ Be honest about risks and limitations
- ✅ Use inclusive, encouraging language

### Don't

- ❌ Promote get-rich-quick schemes
- ❌ Include affiliate links in guides
- ❌ Copy content from other sources without attribution
- ❌ Make unrealistic income claims
- ❌ Share others' personal information
- ❌ Use misleading or clickbait-style content

---

## Review Process

All contributions are reviewed for:

1. **Accuracy** — Are the claims realistic and verifiable?
2. **Completeness** — Does it follow the template and cover all sections?
3. **Quality** — Is the writing clear, actionable, and well-formatted?
4. **Originality** — Is this new content, not duplicated from existing guides?

Reviews typically take 3–7 days.
