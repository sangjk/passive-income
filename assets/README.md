# Assets

This directory contains images, logos, and screenshots used in the PassiveIncome Hub documentation.

## Contents

| File | Description |
|------|-------------|
| `logo.png` | Project logo |
| `framework.png` | 3-Step Passive Income Framework diagram |

## Adding Assets

When adding new images, please:

1. Use descriptive filenames (e.g., `youtube-channel-workflow.png`)
2. Optimize images for web (compress before committing)
3. Keep image sizes under 500KB when possible
4. Use PNG for screenshots and diagrams, JPG for photos
