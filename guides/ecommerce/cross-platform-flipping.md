# 🔄 Cross-Platform Flipping — Side Hustle Guide

## Overview

Buy underpriced items from one platform and resell on another for profit. Requires a good eye for value and knowledge of different marketplaces.

| Detail | Info |
|--------|------|
| **Difficulty** | ⭐⭐⭐ (Medium) |
| **Time to First $** | 1–2 weeks |
| **Earning Potential** | $500–$5,000/month |
| **Upfront Investment** | $50–$500 (inventory) |
| **Best Platforms** | eBay, Facebook Marketplace, Mercari, local thrift stores |

---

## Step 1: Understand the Arbitrage Model

### How Cross-Platform Flipping Works

```
Find undervalued item → Buy low → List on higher-priced platform → Sell high → Profit
```

### Why Price Gaps Exist

| Reason | Example |
|--------|---------|
| **Different audiences** | Vintage camera on eBay (collectors) vs. garage sale (cleaning out) |
| **Poor listing quality** | Bad photos, no description = lower price |
| **Platform fees** | Some platforms have higher fees = higher prices |
| **Urgency** | Sellers who need cash fast price low |
| **Geographic differences** | Items cheaper in one region, expensive in another |
| **Seasonal timing** | Winter gear in summer = cheap; reverse = profit |

## Step 3: Find Deals

### Sourcing Platforms

| Platform | Best For | Average Discount |
|----------|----------|-----------------|
| **Thrift stores** | Clothing, books, home goods | 70–90% off retail |
| **Garage sales** | Everything, especially weekends | 80–95% off retail |
| **Estate sales** | Furniture, jewelry, collectibles | 50–80% off retail |
| **Facebook Marketplace** | Furniture, electronics | 30–60% below market |
| **eBay auctions** | Everything (ending at odd times) | 20–50% below BIN |
| **Craigslist** | Furniture, appliances | 30–50% below market |
| **OfferUp** | Local deals | 30–50% below market |
| **Goodwill online** | Designer items, electronics | 40–70% below market |

### Deal-Finding Strategies

1. **Set alerts** — eBay saved searches, Facebook Marketplace notifications
2. **Shop off-hours** — eBay auctions ending at 2–5 AM = less competition
3. **Look for misspellings** — "Micheal Kors" instead of "Michael Kors"
4. **Bundle buying** — "I'll take all 5 items for $X" = per-item discount
5. **End-of-season** — Buy winter items in spring, sell in fall

### Quick Valuation Check

Before buying, always check:
1. **Completed eBay listings** — What did this item actually sell for?
2. **Current listings** — What's the competition charging?
3. **Condition** — Any flaws that affect value?
4. **Shipping costs** — Will shipping eat your profit?

## Step 4: List & Sell

### Photography Tips

1. **Natural lighting** — Shoot near a window
2. **Clean background** — White sheet or poster board
3. **Multiple angles** — Front, back, sides, details, flaws
4. **Show scale** — Include something for size reference
5. **Highlight condition** — Photograph any wear or damage honestly

### Listing Best Practices

| Element | What to Do |
|---------|-----------|
| **Title** | Brand + model + size + color + condition + keywords |
| **Description** | Honest condition, measurements, history, why you're selling |
| **Price** | Set 10–20% above your target (room for negotiation) |
| **Shipping** | Calculate accurately; offer free shipping if margin allows |
| **Keywords** | Use all relevant search terms |
| **Timing** | List on Thursday–Sunday evenings for best visibility |

### Pricing Formula

```
Listing Price = Purchase Price + Shipping Cost + Platform Fee + Desired Profit

Example:
- Purchase: $10
- Shipping: $8
- Platform fee (13%): ~$5
- Desired profit: $15
- Listing price: $38
```

## Step 5: Scale Your Operation

### Scaling Strategies

1. **Systematize sourcing** — Regular schedule, route planning, alerts
2. **Batch listing** — Create 10–20 listings in one session
3. **Cross-list** — Same item on multiple platforms simultaneously
4. **Build relationships** — Regular sellers give you first pick
5. **Hire help** — Part-time photographer or lister ($10–$15/hour)

### Tools for Scaling

| Tool | Purpose | Cost |
|------|---------|------|
| **Crosslisting apps** | List once, publish everywhere | $10–$30/mo |
| **Price tracking** | Monitor market values | Free–$20/mo |
| **Shipping calculator** | Accurate shipping costs | Free |
| **Inventory tracker** | Track purchases and sales | Free–$15/mo |
| **Photo editing** | Enhance product photos | Free (Canva) |

## ⚠️ Common Mistakes

1. **Not checking completed prices** — Asking price ≠ selling price
2. **Ignoring shipping costs** — Heavy items can have zero profit
3. **Overestimating condition** — Be honest; returns kill margins
4. **Tying up cash in slow items** — Turn inventory in 2–4 weeks max
5. **Not accounting for fees** — Platform fees, payment processing, taxes
