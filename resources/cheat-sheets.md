## 💰 Pricing Quick Reference

### Hourly Rate Benchmarks

| Service | Beginner | Intermediate | Expert |
|---------|----------|-------------|--------|
| **Graphic Design** | $15–$25 | $25–$50 | $50–$150 |
| **Video Editing** | $20–$30 | $30–$60 | $60–$150 |
| **Writing/Copy** | $15–$30 | $30–$60 | $60–$200 |
| **Web Development** | $25–$50 | $50–$100 | $100–$200 |
| **Python/Automation** | $30–$50 | $50–$100 | $100–$200 |
| **Consulting** | $50–$100 | $100–$200 | $200–$500 |
| **Voiceover** | $50–$100/hr | $100–$200/hr | $200–$500/hr |

### How to Calculate Your Rate

```
Rate = (Desired Annual Income + Expenses) ÷ Billable Hours

Example:
  Desired income: $60,000
  Expenses: $10,000
  Billable hours: 1,200/year (25 hrs/week × 48 weeks)
  Rate = ($60,000 + $10,000) ÷ 1,200 = $58/hour
```

## 📈 SEO Quick Reference

### On-Page SEO Checklist

- [ ] Title tag includes primary keyword (50–60 characters)
- [ ] Meta description includes keyword (150–160 characters)
- [ ] URL is short and keyword-rich
- [ ] H1 tag includes primary keyword
- [ ] H2/H3 tags include secondary keywords
- [ ] Image alt text describes the image + keyword
- [ ] Internal links to 2–5 related pages
- [ ] External links to 1–3 authoritative sources
- [ ] Content is 1,500+ words for pillar content
- [ ] Page loads in under 3 seconds
- [ ] Mobile-responsive design

### Keyword Research Quick Guide

| Type | Example | Intent | Competition |

## 🚀 Launch Checklist

### Before Launching Your Side Hustle

- [ ] Define your service/product clearly
- [ ] Identify your target customer
- [ ] Set your pricing (research competitors)
- [ ] Create a portfolio (3–5 samples)
- [ ] Set up a payment method (Stripe/PayPal)
- [ ] Create a service agreement/contract
- [ ] Set up a professional email address
- [ ] Create profiles on 2–3 platforms
- [ ] Write your first 3 outreach messages
- [ ] Track your time and income from day 1

### First Week Action Plan

| Day | Action | Time |
|-----|--------|------|
| **1** | Set up profiles on Fiverr/Upwork | 2 hours |
| **2** | Create portfolio samples | 3 hours |
| **3** | Send 10 outreach messages | 2 hours |
| **4** | Create social media profiles | 1 hour |
| **5** | Send 10 more outreach messages | 2 hours |
| **6** | Create first piece of content | 2 hours |
| **7** | Review results, adjust strategy | 1 hour |
